My Projects

Hotel Website

A fully-functional hotel website that allows users to browse room availability, book reservations, and access hotel services. The project involved designing an intuitive user interface, building a backend with a database to manage room availability and bookings, and ensuring the site is responsive and user-friendly.

Built with HTML, CSS, JavaScript, Python, PostgreSQL, Node.js, ejs

Chatbot for Rail Services

A web-based, intelligent chatbot that finds the cheapest train ticket for journey planning, uses machine learning to predict train delays, and provides information that helps railway staff to handle contingencies.

Built with Python, Flask, MongoDB, HTML, CSS and JavaScript

Multi-User Dungeon Game

A multi-user dungeon (MUD) game developed using Object-Oriented Programming, that allows multiple players to explore a text-based virtual world, engage in combat, and solve puzzles. I designed the game to support a variety of gameplay mechanics such as character customization, inventory management, and dynamic combat systems.

Developed with Python

Detecting Abuse in Care Settings

Detecting abusive language in care homes using natural language processing techniques. This involves dataset development, training and evaluating machine learning models, fine-tuning pre-trained large language models, and developing a web-based interface aimed at non-technical users.

Developed using TensorFlow, Pytorch, NLP, HuggingFace, Neural Networks, LLMs. Deployed using Streamlit
