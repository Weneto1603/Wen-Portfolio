Hotel Website

Project Description

I developed a fully-functional hotel website that allows users to browse room availability, book reservations, and access hotel services. The project involved designing an intuitive user interface, building a backend with a database to manage room availability and bookings, and ensuring the site is responsive and user-friendly.

Key Features

  • Dynamic room availability check and booking system
  • Interactive image galleries of the hotel and rooms
  • Responsive design for mobile, tablet, and desktop users
  • Integration with a backend database for managing reservations

Developed using: HTML, CSS, JavaScript, Python, PostgreSQL, Node.js, ejs
